GitHub Website - how to setup and push code to GitHub using Atom

  1. Create a new repository in GitHub.
  2. In your Mac Finder, create the folder with the files you want to be “pushed” (or uploaded) to the repo. In other words, create a folder you will be using to hold all your project files.
  3. Open terminal and cd to that folder.
  4. Enter the following command line commands (but replace bolded with repo name/url) Note: you will need to press enter for the last command line command if you copy and paste the commands directly into your terminal window:
    5. echo "# examplerepo" >> README.md git init git add README.md git commit -m "first commit" git remote add origin https://github.com/perrysessions/examplerepo.git git push -u origin master
  5. Now you are connect to the GitHub repo with your file on your computer. Woo!
  6. Go to the repository settings and scroll down to GitHub pages. Click the “Choose a theme” button (doesn’t matter which theme because you will be deleting it shortly).
  7. Go to your repo “code” and delete everything other than the readme file.
  8. Go back into terminal and enter the command “git pull —all” this will make your finder folder match what is in the GitHub repo.
  9. Now make a change to the readme file in atom and then use atom’s GitHub integration to stage and commit the changes.
  10. Then go back to terminal and enter “git push” (or git push —all but I think just git push will work). This will push the changes you made to the readme file up to GitHub repo.
  11. You should be able to see a url of your new website in your repo under the location of settings>> scroll to GitHub pages. Sometimes it takes a while to populate that url. Sometimes it is ready right away.
  12. That’s it! Now you can make edits to the code in atom and push it live to your repo when you are ready.
