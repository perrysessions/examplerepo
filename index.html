WELCOME TO EXAMPLE REPO!

  1. Create a new repository in GitHub.
  2. In your Mac Finder, create the folder with the files you want to be “pushed” (or uploaded) to the repo. In other words, create a folder you will be using to hold all your project files.
  3. 3. Open terminal and cd to that folder.
  4. 4. Enter the following command line commands (but replace bolded with repo name/url) Note: you will need to press enter for the last command line command if you copy and paste the commands directly into your terminal window: echo "# examplerepo" >> README.md
    5. git init git add README.md git commit -m "first commit" git remote add origin https://github.com/perrysessions/examplerepo.git git push -u origin master
  5. 7. Now you are connect to the GitHub repo with your file on your computer. Woo!
  6. 8. Go to the repository settings and scroll down to GitHub pages. Click the “Choose a theme” button (doesn’t matter which theme because you will be deleting it shortly). You should be able to see a url of your new website in that same location of settings>>GitHub pages
  8. 9. Delete all the files in the repository after you have picked a theme
